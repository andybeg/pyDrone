# ESP-Дрон

[中文]

ESP-Drone — это решение для дронов с открытым исходным кодом, основанное на чипе Wi-Fi Espressif ESP32/ESP32-S2/ESP32-S3, которым можно управлять через сеть Wi-Fi с помощью мобильного приложения или геймпада. ESP-Drone поддерживает несколько режимов полета, включая режим стабилизации, режим удержания высоты и режим удержания положения. Благодаря простому аппаратному обеспечению, понятной и расширяемой архитектуре кода ESP-Drone можно использовать в образовании STEAM и других областях. Основной код портирован из проекта с открытым исходным кодом Crazyflie с протоколом GPL3.0.

В этом документе описывается руководство по разработке проекта ESP-Drone.
