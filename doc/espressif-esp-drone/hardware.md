## Поддерживаемое оборудование

### Список поддерживаемого оборудования

| Совет по развитию | Основные компоненты | Примечания |
| --- | --- | --- |
| ESP32-S2-Дрон V1.2 | ESP32-S2-WROVER + MPU6050 | Все в одном |
| ESPlane-V2-S2 | ESP32-S2-WROVER + MPU6050 | Необходимо установить четыре ножки |
| ESPlane-FC-V1 | ESP32-WROOM-32D + MPU6050 | Необходимо установить раму дрона. |

### Настройка целевого оборудования

- Код в репозитории esp_drone поддерживает широкий спектр оборудования, которое можно изменить с помощью менюconfig.
- По умолчанию, если для set-target установлено значение esp32s2, целевое оборудование будет автоматически изменено на ESP32_S2_Drone_V1_2.
- По умолчанию, если для set-target установлено значение esp32, целевое оборудование будет автоматически изменено на ESPlane_FC_V1.

Примечания

- ESPlane-FC-V1 — это аппаратная версия старой версии.
- Чтобы использовать код новой версии ESP-Drone на ESPlane-FC-V1, подключите GPIO14 ESP32-WROOM-32D к выводу INT MPU6050 с помощью перемычки.
- Чтобы избежать переключения напряжения флэш-памяти, вызванного IO12, когда ESPlane-FC-V1 включен, зафиксируйте напряжение флэш-памяти на уровне 3,3 В, запустив сценарий espefuse.py espefuse.py --port /dev/ttyUSB0 set_flash_voltage 3,3V Примечание * Только первое устройство, подключенное к шине, может использовать контакт CS0.
